# SEO Checklist for Adding New Products

## When Adding New Products to Kalangan

### 1. Product Name
- ✅ Use descriptive, keyword-rich names
- ✅ Include product type (e.g., "Customized Wedding Frame" not just "Frame")
- ✅ Add occasion if relevant (e.g., "Anniversary Gift Nameplate")

**Examples:**
- ✅ Good: "Personalized Wooden Nameplate for Home"
- ❌ Bad: "Nameplate 001"

### 2. Product Description
Include these elements:
- ✅ What it is (product type)
- ✅ Who it's for (target audience)
- ✅ When to use it (occasions)
- ✅ Key features (handmade, customizable, etc.)
- ✅ Materials used
- ✅ Size/dimensions

**Template:**
```
[Product Type] perfect for [Occasion]. Handmade with [Materials] in Kalyan, Maharashtra. 
Fully customizable with [Customization Options]. Ideal for [Target Audience]. 
Available with Pan India delivery.
```

**Example:**
```
Customized wedding frame perfect for couples celebrating their special day. Handmade with 
premium wood and acrylic in Kalyan, Maharashtra. Fully customizable with names, dates, 
and photos. Ideal for wedding gifts, anniversary presents, or home decoration. 
Available with Pan India delivery.
```

### 3. Product Images
- ✅ Use descriptive file names: `wedding-frame-wooden-personalized.jpg`
- ❌ Avoid: `IMG_1234.jpg`
- ✅ Add alt text: "Customized wooden wedding frame with couple names"
- ✅ Optimize image size (compress for web)

### 4. Product Categories
Assign to relevant categories:
- Wedding Frames
- Nameplates
- Birthday Gifts
- Customized Frames
- Fridge Magnets
- Keychains

### 5. Keywords to Include
For each product, think about:
- **Product type**: wedding frame, nameplate, keychain
- **Material**: wooden, acrylic, metal
- **Occasion**: wedding, birthday, anniversary
- **Customization**: personalized, customized, with names
- **Location**: Kalyan, Maharashtra, India

### 6. Pricing Strategy
- ✅ Show price if standard product
- ✅ Use "Contact for price" if fully customized
- ✅ Mention bulk order discounts in description

### 7. Tags/Labels
Add relevant tags:
- Handmade
- Customizable
- Best Seller
- New Arrival
- Wedding Special
- Return Gift
- Eco-friendly

## Quick SEO Tips

### DO ✅
- Use natural, conversational language
- Include location (Kalyan, Maharashtra)
- Mention Pan India delivery
- Highlight customization options
- Use occasion-based keywords
- Add material details
- Include "handmade" and "personalized"

### DON'T ❌
- Stuff keywords unnaturally
- Use generic descriptions
- Forget alt text for images
- Skip product categories
- Use only product codes as names
- Ignore mobile optimization

## Keyword Research for New Products

Before adding a new product type, research:
1. Google search volume for related terms
2. Competitor product names
3. Customer search queries
4. Trending gift ideas
5. Seasonal demands

**Tools to Use:**
- Google Keyword Planner
- Google Trends
- Answer The Public
- Customer feedback/requests

## Updating Existing Products

When updating products:
- ✅ Refresh descriptions with new keywords
- ✅ Add better quality images
- ✅ Update pricing if changed
- ✅ Add customer reviews
- ✅ Include seasonal relevance

## Seasonal SEO Optimization

### Wedding Season (Oct-Feb)
- Highlight wedding frames, nameplates
- Use keywords: "wedding return gifts", "couple gifts"

### Birthday Season (Year-round)
- Promote birthday frames, keychains
- Keywords: "birthday gift ideas", "personalized birthday presents"

### Festival Season (Aug-Nov)
- Focus on return gifts, fridge magnets
- Keywords: "festival gifts", "Diwali return gifts"

### Housewarming Season (Jan-Mar)
- Emphasize nameplates, frames
- Keywords: "housewarming gifts", "new home gifts"
